Jmeter参数化
==========================================

CSV Data Set Config
--------------------------------------

CSV Data Set Config可以制定文本文件这一行一行的提取文本内容，跟进分割符拆解这一行内容与变量名对应，然后这些变量就可以提供给取样器引用了。

参数说明如下：
名称：随意设置，最好有业务意义
注释：随意设置，可以为空
Filename：应用文件地址，可以是相对路径，也可以是绝对路径。相对路径的根节点是Jmeter的启动文件路径（%JMETER_HOME%\bin）
File encoding：读取参数文件的编码格式，建议使用UTF-8
